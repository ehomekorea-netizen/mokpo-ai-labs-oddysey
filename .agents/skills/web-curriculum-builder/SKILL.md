# Web Curriculum Builder Skill

이 기술은 수강생이 웹 개발 개념을 시각적으로 학습하고 실습할 수 있도록 설계된 **'프리미엄 1페이지 대화형 교안 사이트'**를 설계, 확장 및 빌드하는 기술입니다. 특정 프로젝트에 종속되는 변수(예: 비디오 슬라이서 등)는 옵션으로 격리하고, 변하지 않는 **디자인 정체성**과 **대화형 플레이그라운드 기본 스펙**을 규격화하여 범용적으로 사용합니다.

---

## 1. 디자인 정체성 표준 (Aesthetic Guardrails - 고정 규칙)

모든 교안 페이지는 아날로그의 따뜻한 감성과 현대적인 에디토리얼 무드를 극대화하여 첫인상부터 프리미엄 브랜드의 가치가 느껴지도록 톤앤매너를 유지해야 합니다.

*   **배경색 (Background)**: 따뜻하고 부드러운 종이 미색 (`bg-[#f8f4ed]`) - FOUC 방지를 위해 `<body>` 인라인 스타일에 고정 적용.
*   **텍스트 주색상 (Text)**: 에디토리얼 감성의 오프블랙/잉크색 (`text-[#191f28]` 또는 `text-ink-900`)
*   **타이포그래피 스케일**: default 브라우저 서체를 배제하고 구글 폰트(Outfit, Inter 등)를 탑재.
    *   헤드라인/대형 타이틀: 자간이 극도로 좁은(tracking-tighter) 볼드 서체.
    *   본문/해설: 가독성이 뛰어난 16px~18px(`text-base` 내지 `text-lg`) 구성 및 여유로운 자간/행간.

---

## 2. 동적 인트로 및 플레이그라운드 설계 (Core Architecture)

### A. 동적 인트로 & 실습실 입장 모달 (Dynamic Lab Entrance)
*   **작동 규칙**: 특정 수업 명칭이 하드코딩되지 않고, 사용자가 지정한 **'수업 주제(Topic)'**를 AI가 자동으로 인식하여 인트로 모달의 타이틀과 설명문을 동적으로 커스터마이징합니다.
*   **UI 구성**:
    *   초입에 눈을 사로잡는 대형 환영 메시지 및 핵심 비주얼 카드.
    *   **실습실 입장하기(Enter the Lab)** 동작을 유도하는 직관적인 진입 버튼 배치.
    *   수업 명세가 자동으로 반영된 소개글 제공.

### B. 웹 기본 플레이그라운드 위젯 (Step 0 - Default Simulators)
수업 주제와 관계없이 프론트엔드의 기초 원리를 깨달을 수 있도록 돕는 3대 기본 위젯 세트입니다.
1.  **인터넷 vs 웹, 그리고 배포 (동적 패킷 트뮬레이터)**: URL 요청 시 가상 패킷이 클라이언트와 서버를 오가며 렌더링을 트리거하는 애니메이션.
2.  **HTML 뼈대와 CSS 옷 (실시간 스타일 토글러)**: 스타일 스위치 작동 시 꾸밈 없는 기본 뼈대 서체와 세련된 Tailwind 카드가 전환되는 대화형 뷰어.
3.  **JS: 브라우저 리모컨 (이벤트 & 콘솔 로거)**: 버튼 액션 시 작동 원리 코드를 단계별로 하이라이트해 주는 가상 개발자 콘솔.

---

## 3. 프로젝트별 가변적 위젯 블록 (Project-Specific Variable Blocks)

*   **비디오 스크롤 슬라이서 (Step 7 등)**:
    *   비디오를 프레임 단위로 추출하는 슬라이서와 같은 컴포넌트는 모든 교안에 공통으로 들어가는 기본 스펙이 아닙니다.
    *   수업 주제가 '영상 인터랙션'일 경우에만 **선택적 프로젝트 변수 블록**으로 추가/연동하며, 다른 주제(예: 프롬프트 엔지니어링, 챗봇 제작 등)로 변형될 때는 해당 주제에 맞는 특화 위젯으로 유연하게 대체 가능하도록 설계합니다.

---

## 4. 코드 구현 및 안전 규정

1.  **단일 파일 배포성 (Single HTML File)**: 모든 자바스크립트 비주얼 모션 로직과 CSS 스타일링은 별도의 파일 분리 없이 `index.html` 파일 하나에 자생적으로 작동할 수 있도록 결합합니다.
2.  **안전 경로 이동 자동화**: 에이전트가 터미널 명령어를 안내할 때는 사용자의 경로 이탈을 막기 위해 반드시 첫 줄에 해당 프로젝트 경로(`cd C:\Users\IN\Desktop\mokpo-ai-curriculum`)를 자동으로 삽입합니다.
