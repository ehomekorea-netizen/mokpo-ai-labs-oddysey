# Web Curriculum Builder Skill

이 기술은 수강생이 특정 학습 주제의 핵심 개념을 브라우저상에서 시각적으로 조작하며 직관적으로 이해할 수 있도록 돕는 **'프리미엄 1페이지 대화형 교안 사이트'**를 설계 및 빌드하는 가이드입니다.

---

## 1. 스킬의 핵심 목적 (Core Purpose)

*   **입력 데이터**: 강사님이 제공하는 **`project.md`** (교육 목적, 대상자, 커리큘럼 계획, 아키텍처 및 핵심 이론이 담긴 기획 명세서).
*   **에이전트의 역할**:
    1.  `project.md`의 내용을 깊이 있게 분석하여 **가장 중요하고 추상적인 개념(아키텍처, 작동 흐름 등)**을 발굴합니다.
    2.  이 핵심 개념을 텍스트 설명에 그치지 않고, 수강생들이 직접 조작하며 동작 원리를 즉시 깨달을 수 있는 **'신선하고 인터랙티브한 시각화 플레이그라운드(Step 0)'**로 재창조합니다.
    3.  이를 통해 교육을 진행하는 강사나 참여하는 수강생 모두에게 **비주얼적인 몰입감과 실시간 반응형 피드백**을 제공하는 것을 궁극적인 목적으로 삼습니다.

---

## 2. 디자인 정체성 표준 (Aesthetic Guardrails - 고정 규칙)

모든 교안 페이지는 아날로그의 따뜻한 감성과 현대적인 에디토리얼 무드를 극대화하여 첫인상부터 프리미엄 브랜드의 가치가 느껴지도록 톤앤매너를 유지해야 합니다.

*   **배경색 (Background)**: 따뜻하고 부드러운 종이 미색 (`bg-[#f8f4ed]`) - FOUC 방지를 위해 `<body>` 인라인 스타일에 고정 적용.
*   **텍스트 주색상 (Text)**: 에디토리얼 감성의 오프블랙/잉크색 (`text-[#191f28]` 또는 `text-ink-900`)
*   **타이포그래피 스케일**: default 브라우저 서체를 배제하고 구글 폰트(Outfit, Inter, Noto Sans KR 등)를 탑재.
    *   헤드라인/대형 타이틀: 자간이 극도로 좁은(tracking-tight 또는 tracking-tighter) 볼드 서체.
    *   본문/해설: 가독성이 뛰어난 16px~18px(`text-base` 내지 `text-lg`) 구성 및 여유로운 자간/행간.
*   **빔프로젝터 가독성 (Projector-Friendly Typography)**:
    *   교안을 화면에 띄우고 라이브 세미나 형식으로 강의하기 위해 모든 글씨 크기를 기본 폰트 대비 1.2배 이상 크게 구성합니다.
    *   컨테이너 주변의 여백(Padding)과 화이트스페이스를 넓게 설정하여 멀리서도 텍스트 줄바꿈이 한눈에 보이게 넉넉히 배치합니다.

---

## 3. 프리미엄 인터랙티브 디자인 패턴 (Premium Interactive Patterns)

검증된 프론트엔드 핵심 UI 모션과 데이터 구조를 교안 사이트에 흡수하여 인터랙션 완성도를 극대화합니다.

### A. 마우스 X축 트래킹 세로 SVG 라인 (Scroll-Linked / Mouse Tracking Index)
*   **구조**: 사용자의 마우스 X축 좌표를 추적하여 세로 점선(`line`)과 노드 포인트(`circle`)가 부드럽게 좌우로 따라 움직이는 비주얼 인디케이터.
*   **의도**: 마우스 포인터의 위치에 따라 정보가 실시간으로 갱신되거나 초점을 맞추는 동적인 느낌을 제공합니다.
*   **구현**: SVG 요소의 `x1`, `x2`, `cx` 속성을 자바스크립트 `onmousemove` 이벤트를 받아 제어합니다.

### B. 미니멀 레저 그리드 (Typography Ledger Grid)
*   **구조**: 1px 두께의 초미세 연한 테두리 경계(`border-black/5`)를 가지는 미니멀한 격자 형태의 표 레이아웃.
*   **의도**: 셀에 마우스 호버 시 배경색과 글자색이 순식간에 반전(`hover:bg-[#191f28] hover:text-white transition-colors duration-150`)되는 물리 피드백을 주어 고급 모바일 앱의 햅틱 피드백 무드를 연출합니다.

### C. 1:1 프롬프트 추론/번역 카드 (Vibe 1:1 Translation Cards)
*   **구조**: 초급자의 '날것 그대로의 자연어 요청'과 '추론 모델의 기술적 해석(CORS, Proxy, Web Socket 등)', 그리고 최종 변환된 '정밀 기술 지시 프롬프트'를 한눈에 볼 수 있는 가로 병렬형 대조 카드 블록.
*   **의도**: 코딩을 모르는 비개발자도 자신의 모호한 요구가 AI에 의해 구체적인 아키텍처로 어떻게 추론 및 구현되었는지 역추적하여 학습하도록 돕습니다.

### D. 특수 효과 및 버튼 햅틱 피드백 (Micro-Interactions)
*   **디코더 타이틀 (Text Scramble)**: 페이지 접속이나 호버 시 글자가 무작위 기호로 변하며 깜빡거리다 원래 글로 복구되는 모션.
*   **필름 노이즈 배경 (Film Grain)**: 투명하고 미세하게 진동하는 배경 오버레이를 씌워 아날로그적인 편안함 연출.
*   **탄성 바운스 버튼 (Elastic Bounce)**: 마우스 클릭/터치 다운 시 젤리처럼 탱글하게 스케일이 벼락치듯 작아졌다 커지는 모션 (`active:scale-[0.97] transition-all`).

## 4. 프로젝트별 가변적 위젯 블록 (Project-Specific Variable Blocks)

*   수업 주제가 '영상 디자인/인터랙션'일 때 추가되는 **비디오 스크롤 슬라이서**와 같이 특정 실습 도구가 필요할 때만 선택적으로 해당 단계(Step 7 등)에 플러그인 형태로 추가/대체하도록 설계합니다.

---

## 5. 코드 구현 및 안전 규정

1.  **단일 파일 배포성 (Single HTML File)**: 모든 자바스크립트 비주얼 모션 로직과 CSS 스타일링은 별도의 파일 분리 없이 `index.html` 파일 하나에 자생적으로 작동할 수 있도록 결합합니다.
2.  **안전 경로 이동 자동화**: 에이전트가 터미널 명령어를 안내할 때는 사용자의 경로 이탈을 막기 위해 반드시 첫 줄에 해당 프로젝트 경로(`cd C:\Users\IN\Desktop\mokpo-ai-curriculum`)를 자동으로 삽입합니다.
3.  **스크립트 태그 및 구문 무결성 (Script Tag & Syntax Integrity)**: HTML 스크립트 블록 내에서 코드를 수정하거나 덮어쓸 때, `</script>` 태그가 잘려 깨지는 현상(예: `</script //...`)이나 닫히지 않은 중괄호, 중복 변수 선언 등으로 인해 브라우저 파서가 작동 불능에 빠지지 않도록 각별히 유의합니다. 코드 치환 후에는 반드시 전체 스크립트 컴파일 테스트를 거쳐 `closeSplash` 등 핵심 UI 함수가 정상 작동하는지 재차 검증합니다.
